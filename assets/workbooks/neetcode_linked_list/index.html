Exercise 1: Reverse Linked List

Basic Pointer Manipulation

Objective

Reverse a singly linked list iteratively in O(n) time, O(1) space.

Visual Example

Original: 1 â†’ 2 â†’ 3 â†’ 4 â†’ 5 â†’ None Step-by-step: prev curr next None 1 â†’ 2 â†’ 3 â†’ 4 â†’ 5 â†’ None Reverse: 1.next = None None â† 1 2 â†’ 3 â†’ 4 â†’ 5 â†’ None prev curr next 1 â† 2 3 â†’ 4 â†’ 5 â†’ None None â† 1 â† 2 3 â†’ 4 â†’ 5 â†’ None ... Result: 5 â†’ 4 â†’ 3 â†’ 2 â†’ 1 â†’ None â†‘ head

Key Pattern

def reverseList(head: ListNode) -> ListNode:
    prev = None
    curr = head

    while curr:
        # Save next node
        next_node = curr.next
        # Reverse link
        curr.next = prev
        # Move pointers forward
        prev = curr
        curr = next_node

    return prev  # New head

Time: O(n) | Space: O(1)

Exercise 2: Merge Two Sorted Lists

Dummy Node Pattern

Objective

Merge two sorted linked lists into one sorted list.

ğŸ’¡ Dummy Node Trick

Use a dummy node to avoid special-casing the first element. Build the result list by comparing values and attaching the smaller node.

Visual Example

list1: 1 â†’ 3 â†’ 5 list2: 2 â†’ 4 â†’ 6 dummy â†’ (start building result here) Compare 1 vs 2: attach 1 dummy â†’ 1 Compare 3 vs 2: attach 2 dummy â†’ 1 â†’ 2 Compare 3 vs 4: attach 3 dummy â†’ 1 â†’ 2 â†’ 3 ... Result: 1 â†’ 2 â†’ 3 â†’ 4 â†’ 5 â†’ 6

Key Pattern

def mergeTwoLists(list1: ListNode, list2: ListNode) -> ListNode:
    dummy = ListNode(0)
    tail = dummy

    while list1 and list2:
        if list1.val < list2.val:
            tail.next = list1
            list1 = list1.next
        else:
            tail.next = list2
            list2 = list2.next
        tail = tail.next

    # Attach remaining nodes
    tail.next = list1 if list1 else list2

    return dummy.next

Time: O(n + m) | Space: O(1)

Exercise 3: Linked List Cycle

Floyd's Fast/Slow Pointers

Objective

Detect if a linked list has a cycle using Floyd's algorithm.

ğŸ’¡ The Race Track Analogy

Imagine two runners on a circular track. If one runs twice as fast, they will eventually meet if there's a cycle. If there's no cycle, the fast runner reaches the end (None).

Visual Example

List with cycle: 3 â†’ 2 â†’ 0 â†’ -4 â†‘________â†“ slow: 3 â†’ 2 â†’ 0 â†’ -4 â†’ 2 â†’ 0 â†’ ... fast: 3 â†’ 0 â†’ 2 â†’ -4 â†’ 0 â†’ ... â†‘ They meet! Cycle detected List without cycle: 1 â†’ 2 â†’ 3 â†’ None slow: 1 â†’ 2 â†’ 3 â†’ None fast: 1 â†’ 3 â†’ None (fast reaches None first)

Key Pattern

def hasCycle(head: ListNode) -> bool:
    slow = fast = head

    while fast and fast.next:
        slow = slow.next       # Move 1 step
        fast = fast.next.next  # Move 2 steps

        if slow == fast:
            return True  # Cycle detected

    return False  # fast reached None

Time: O(n) | Space: O(1)

Exercise 4: Reorder List

Multi-Step Algorithm

Objective

Reorder list L0â†’L1â†’...â†’Ln to L0â†’Lnâ†’L1â†’Ln-1... in-place.

ğŸ’¡ Three-Step Strategy

1) Find middle using slow/fast pointers 2) Reverse second half 3) Merge two halves alternately

Visual Example

Input: 1 â†’ 2 â†’ 3 â†’ 4 â†’ 5 Step 1: Find middle (slow/fast) 1 â†’ 2 â†’ 3 4 â†’ 5 â†‘ â†‘ first second Step 2: Reverse second half 1 â†’ 2 â†’ 3 5 â†’ 4 Step 3: Merge alternately 1 â†’ 5 â†’ 2 â†’ 4 â†’ 3

Key Pattern

def reorderList(head: ListNode) -> None:
    # Step 1: Find middle
    slow = fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next

    # Step 2: Reverse second half
    second = slow.next
    slow.next = None
    prev = None
    while second:
        next_node = second.next
        second.next = prev
        prev = second
        second = next_node
    second = prev

    # Step 3: Merge alternately
    first = head
    while second:
        tmp1, tmp2 = first.next, second.next
        first.next = second
        second.next = tmp1
        first, second = tmp1, tmp2

Time: O(n) | Space: O(1)

Exercise 5: Remove Nth Node From End

Two-Pointer with Gap

Objective

Remove the nth node from the end of the list in one pass.

ğŸ’¡ The Gap Trick

Maintain two pointers with n-node gap between them. When fast reaches the end, slow is at (n-1)th from endâ€”perfect position to remove nth node.

Visual Example

Remove 2nd from end: 1 â†’ 2 â†’ 3 â†’ 4 â†’ 5, n=2 Step 1: Move fast n steps ahead dummy â†’ 1 â†’ 2 â†’ 3 â†’ 4 â†’ 5 â†’ None â†‘ â†‘ slow fast Step 2: Move both until fast reaches end dummy â†’ 1 â†’ 2 â†’ 3 â†’ 4 â†’ 5 â†’ None â†‘ â†‘ slow fast Step 3: Remove: slow.next = slow.next.next Result: 1 â†’ 2 â†’ 3 â†’ 5

Key Pattern

def removeNthFromEnd(head: ListNode, n: int) -> ListNode:
    dummy = ListNode(0, head)
    slow = fast = dummy

    # Move fast n steps ahead
    for _ in range(n):
        fast = fast.next

    # Move both until fast reaches end
    while fast.next:
        slow = slow.next
        fast = fast.next

    # Remove nth node
    slow.next = slow.next.next

    return dummy.next

Time: O(n) | Space: O(1)

Linked List Patterns Summary

1. Pointer Patterns

  • Single pointer: Simple traversal
  • Prev/curr/next: Reversal, deletion
  • Slow/fast: Find middle, detect cycle
  • Two with gap: Nth from end
  • Dummy node: Simplify head handling

2. Common Techniques

# Find middle
slow = fast = head
while fast and fast.next:
    slow = slow.next
    fast = fast.next.next
# slow is at middle

# Reverse list
prev, curr = None, head
while curr:
    next_node = curr.next
    curr.next = prev
    prev, curr = curr, next_node
# prev is new head

# Dummy node
dummy = ListNode(0)
dummy.next = head
# ... work with dummy.next ...
return dummy.next

3. Edge Cases to Always Check

  • Empty list: head = None
  • Single node: head.next = None
  • Two nodes: Test your pointer logic
  • Cycles: Infinite loop risks
